# DevFlow - Modern Software Development Agency

A modern, animated website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## 🚀 Features

- Responsive design with modern UI/UX
- Smooth animations and transitions
- Service showcase
- Contact form
- Performance optimized
- SEO friendly

## 🛠️ Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel Deployment

## 🏃‍♂️ Getting Started

1. Clone the repository:
```bash
git clone https://github.com/jsa498/DevFlow.git
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
